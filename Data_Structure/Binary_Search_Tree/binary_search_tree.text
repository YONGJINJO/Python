[이진 탐색 트리]
  모든 왼쪽에 있는 노드는 자기자신보다 작아야 한다.
  모든 오른쪽에 있는 노드는 자기자신보다 커야 한다.

  파이썬 리스트가 아닌 노드 클래스를 이용하여 구현

  [이진탐색트리 출력]
    in-order 출력 사용

  [이진탐색트리 삽입]
    삽입이 끝나도 속성이 유지되어야 한다.
  [이진탐색트리 삭제]
    경우 1 : 지우려는 노드가 leaf 노드인 경우
      해당 노드만 지우면 된다
    경우 2 : 지우려는 노드가 하나의 자식만 있는 경우
      지우려는 노드의 부모노드와 자식노드를 연결하면 된다.

[데이터 만들기의 중요성]

센서(자극을 전기 신호로 바꿔주는 것) 사용하기
아두이노/라즈베리파이를 이용해 센서를 이용한 데이터 활용 가능
but 가장 빠른 속도로 데이터가 쌓이는 곳은 인터넷
따라서 웹에서 데이터를 모으는 것이 가장 쉬움

데이터 수집
- 웹 스크레이핑 : 하나의 특정 웹페이지에서 원하는 정보를 받아오는 것
- 웹 크롤링 : 컴퓨터가 자동으로 여러 웹페이지를 스크레이핑하게 하는 것

웹사이트 주소 이해하기

https -> 소통 방식
www.ikea.com -> 도메인 이름
catalog/news -> 경로
sorting=price... -> 쿼리스트링(파라미터)

웹피이지 스타일링 -> CSS


파싱 : 문자의 구조를 분석해서 원하는 정보를 뽑아내는 것
